Nginx to serwer odwrotnego proxy typu open source dla protokołów HTTP, HTTPS, SMTP, POP3 i IMAP, a także system równoważenia obciążenia, HTTP cache i serwer WWW. Projekt nginx rozpoczął się od silnego skupienjia się na wysokiej współbieżności, wysokiej wydajności i niskim zużyciu pamięci. Posiada również port dowodu koncepcji dla systemu Microsoft Windows.
Nginx podłączony do inteneru na porcie 80.
Gdy żądanie jest wysyłane do dowolnego punktu końcowegi frontendu, serwer frontend express obsługuje statuczne strony internetowe html.
Gdy żądanie jest wysyłane ze strony internetowej gs_calc, serwer sieciowy api express obsługuje te żądania.
